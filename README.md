# KCCI OnDevice AI 1st

본 프로젝트는 한국상공회의소에서 진행하는 OnDevice AI 1기 교육 프로그램의 결과물입니다.

## 🚀 시작하기

### 필수 요구사항
- Python 3.8 이상
- 필요한 라이브러리 설치

### 설치 방법

```bash
# 저장소 클론
git clone https://github.com/hochaeAidl/KCCI-Ondevice-AI-1st.git
cd KCCI-Ondevice-AI-1st

# 필수 패키지 설치
pip install -r requirements.txt
```
